# 任务模板索引

本目录包含 code-task 阶段输出的任务文档模板。

## 模板列表

| 模板文件 | 用途 | 场景 |
|---------|------|------|
| [`task-template.md`](./task-template.md) | 单个Task定义模板 | 所有场景 |
| [`tasks-document-template.md`](./tasks-document-template.md) | 完整tasks.md文档模板 | 所有场景 |
| [`frontend-tasks-guide.md`](./frontend-tasks-guide.md) | 前端基础设施任务指南 | 前端/全栈场景 |

## 使用方式

在 `code-task` SKILL.md 的步骤6中，使用这些模板生成 `spec-dev/{requirement_desc_abstract}/tasks/tasks.md`。
