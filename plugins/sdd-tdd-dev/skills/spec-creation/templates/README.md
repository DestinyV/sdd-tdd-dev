# 规范模板索引

本目录包含 spec-creation 阶段输出的各类规范文档模板。

## 模板列表

| 模板文件 | 用途 | 场景 |
|---------|------|------|
| [`spec-requirement-template.md`](./spec-requirement-template.md) | 需求文档总览 | 所有场景 |
| [`spec-scenario-template.md`](./spec-scenario-template.md) | BDD场景定义 | 所有场景 |
| [`spec-data-models-template.md`](./spec-data-models-template.md) | 数据模型定义 | 所有场景 |
| [`spec-business-rules-template.md`](./spec-business-rules-template.md) | 业务规则和约束 | 所有场景 |
| [`spec-glossary-template.md`](./spec-glossary-template.md) | 术语表 | 所有场景 |
| [`spec-infrastructure-template.md`](./spec-infrastructure-template.md) | 基础设施分析（后端） | 后端/全栈场景 |
| [`spec-deployment-template.md`](./spec-deployment-template.md) | 上线和部署方案（后端） | 后端/全栈场景 |
| [`spec-frontend-deployment-template.md`](./spec-frontend-deployment-template.md) | 前端上线和部署方案 | 前端/全栈场景 |
| [`spec-frontend-monitoring-template.md`](./spec-frontend-monitoring-template.md) | 前端监控和可观测性 | 前端/全栈场景 |

## 使用方式

在 `spec-creation` SKILL.md 的阶段4中，根据需求类型选择对应模板生成规范文档。每个模板都有完整的字段定义和示例，可直接填充使用。
